# How to use it
1. Install Python 3.7< and all the dependencies from file requirements.txt
    ``` $ pip install -r requirements.txt ```

2. Run flask app use terminal: 
    - First you must export the FLASK_APP environment variable
        On UNIX
        ```$ export FLASK_APP=app.py```
        On Windows
         ```> SET FLASK_APP=app.py```
    - run flask
    ```$ flask run``` or ```$ python -m flask run```
3. Server is ready
